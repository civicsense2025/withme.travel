# App Layout Components

This section contains components used for organizing and structuring the application layout.

## Navigation Components

- **Navbar**: Main navigation bar at the top of the application
- **Sidebar**: Secondary navigation menu for section-specific links
- **Footer**: Page footer with links and information
- **Breadcrumbs**: Path-based navigation showing the current location

## Layout Components 

- **Container**: Wrapper component for constraining content width
- **Grid**: Responsive grid system for layout
- **Section**: Visual section with optional background and padding
- **Divider**: Horizontal or vertical dividing line

## Page Structures

- **DashboardLayout**: Standard layout for dashboard pages
- **SettingsLayout**: Layout for settings and configuration pages
- **AuthLayout**: Centered layout for authentication pages
- **MarketingLayout**: Full-width layout for marketing pages

## Usage Guidelines

1. Use consistent layouts across similar page types
2. Utilize responsive layouts that work across device sizes
3. Keep navigation elements consistent and predictable
4. Consider white space and content density for readability
