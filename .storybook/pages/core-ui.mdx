# Core UI Components

These foundational UI components serve as building blocks for all interfaces within the withme.travel platform.

## Input Controls

- **Button**: Primary, secondary, and tertiary action buttons
- **Input**: Text fields, textareas, and other input types  
- **Select**: Dropdown selection menus
- **Checkbox**: Single and grouped checkboxes
- **Radio**: Radio button groups
- **Switch**: Toggle switches for binary options

## Display Components

- **Card**: Container for related content and actions
- **Badge**: Small status indicators and counters
- **Avatar**: User profile images and initials
- **Icon**: System iconography and visual indicators
- **Tag**: Keyword and category labeling

## Feedback Components

- **Alert**: Notification messages at various severities
- **Toast**: Temporary notification messages
- **Progress**: Loading indicators and progress bars
- **Skeleton**: Content loading placeholders

## Navigation Components

- **Tabs**: Content section navigation
- **Pagination**: Page navigation for multi-page content
- **Breadcrumb**: Path-based navigation
- **Menu**: Dropdown and context menus

## Usage Guidelines

1. Prefer standard components over custom implementations
2. Maintain consistent styling and behavior
3. Follow accessible design patterns
4. Compose complex interfaces from simpler components

Check out each component's documentation page for detailed usage examples and API references.
