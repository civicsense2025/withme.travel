# withme.travel Design System & Component Library

Welcome to the withme.travel design system and component library. This Storybook instance provides a comprehensive showcase of our UI components, design patterns, and guidelines for building consistent, accessible, and beautiful user experiences.

## Navigation

Our component library is organized into five main sections:

- [**Design System**](?path=/docs/design-system-index--docs): Core foundations - colors, typography, spacing, theme
- [**Core UI**](?path=/docs/core-ui-index--docs): Reusable UI components (Button, Input, Card, etc.)
- [**Features**](?path=/docs/features-index--docs): Feature-specific components (TripCard, ItineraryCard, etc.)
- [**Product Marketing**](?path=/docs/product-marketing-index--docs): Components for marketing/landing pages
- [**App Layout**](?path=/docs/app-layout-index--docs): Navigation, layouts, and containers

## Getting Started

1. Browse components using the sidebar navigation
2. Explore variants and properties using the controls panel
3. View component documentation using the "Docs" tab
4. Check component code using the "Show code" button
