# Marketing Components

These components are specifically designed for landing pages, product marketing, and promotional content.

## Hero Components

- **MainHero**: Primary hero section for homepage
- **FeatureHero**: Highlighted feature showcase
- **TestimonialHero**: Customer testimonial spotlights
- **SignupHero**: Conversion-focused hero with signup form

## Content Sections

- **FeatureGrid**: Grid layout for feature highlights
- **BenefitsSection**: Value proposition display
- **PricingTable**: Plan comparison and pricing
- **TestimonialCarousel**: Scrolling customer testimonials

## Call-to-Action Components

- **CTAButton**: High-visibility action buttons
- **SignupForm**: User registration form
- **NewsletterSignup**: Email collection with benefits
- **ContactForm**: User inquiry form

## Decorative Elements

- **AnimatedIllustration**: Motion graphics for visual appeal
- **BackgroundWave**: Stylized section dividers
- **FloatingElements**: Decorative floating shapes
- **IconGrid**: Feature icon presentation

## Usage Guidelines

1. Maintain consistent branding and messaging
2. Optimize for conversion at key decision points
3. Ensure mobile-friendly layouts and interactions
4. Keep page load performance in mind
5. A/B test different layouts for effectiveness
